# API_PLANO

API desenvolvida para vaga de desenvolvedor Back-end

- Calcula preço do plano de acordo com os dados inseridos
- Registra os dados do usuário

# Tecnologias utilizadas

-Php
- Composer
- Javascript
- Html
- Css
# Rodando a aplicação

É necessário possuir as seguintes dependências:

- Php 5 ou superior
- Composer
